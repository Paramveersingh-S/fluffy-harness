## [1.0.0] - 2026-06-15
### Added
- Initial release of pi-guard extension.
- Granular bash regex matching.
- File path glob matching for write tools.
- Strict JSONL audit logging with credential redaction.
- Headless fail-closed mechanics for `confirm` actions.
